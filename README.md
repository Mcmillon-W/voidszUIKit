# voidszUIKit
Just a Util Library helpful for android app development

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
Add the dependency:
  
  ```
 dependencies {
   implementation 'com.github.Mcmillon-W:voidszUIKit:v1.0.1'
}
```
